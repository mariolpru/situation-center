## Workflow MCP

- Сначала `get_metadata` (структура), потом `get_design_context` на подузлы (большие узлы падают по токенам).
- Скриншоты браузер-панели запаздывают — проверять состояние через `javascript_tool` (но! панель блокирует программные inline-стили — hover проверять физическим hover-действием).
- SVG из Figma приходят с `width="100%"` — прописывать явные размеры из viewBox (иначе `<img>` едет).
